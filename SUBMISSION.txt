SYSTEM OVERVIEW
--------------------------------------------------

This project builds an AI-powered Carbon Footprint Tracking system that:

• Allows users to log daily activities (transport, energy, diet)  
• Calculates CO₂ emissions using standardized emission factors  
• Stores and manages activity data securely  
• Provides emission breakdown by category  
• Visualizes trends using interactive charts  
• Generates AI-powered sustainability recommendations  
• Compares daily, weekly, and monthly emission patterns  

--------------------------------------------------
AI COMPONENT
--------------------------------------------------

Generative AI Recommendations → OpenAI / Gemini API  

AI is used to:

• Analyze user emission patterns  
• Identify high-impact lifestyle areas  
• Generate personalized sustainability advice  
• Translate emission data into actionable eco-friendly suggestions
